# SP AUTO KILL-PHASE

Auto cast/movement for SP mage boost kill phase in World of Warcraft TBC

## Gather the mobs at the kill spot and let the bot do the work

Set keybinds:
"1" - Blizzard (rank1)
"5" - Flamestrike (rank1)
"5" - Flamestrike
"0" - Create Mana Gem
"." - Use Mana Gem
